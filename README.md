# DermHandouts

A simple GitHub Pages site for dermatology patient handouts (Acne, Accutane/Isotretinoin).

## Enable GitHub Pages

1. Open your repo on GitHub → **Settings** → **Pages**.
2. Under **Source**, choose **Deploy from a branch**.
3. Branch: **main**, folder: **/ (root)**.
4. Save. The site will be at `https://<username>.github.io/DermHandouts/`.
